== Installation ==

# DEPUIS VOTRE COMPTE ADMIN WORDPRESS

Sur https://votre-site.fr/wp-admin

1. Allez sur Extensions > Ajouter
2. Cherchez Stan Checkout, trouvez et installez Stan Xpress Checkout.
3. Activez le plugins depuis la page Extensions

# DEPUIS WORDPRESS.ORG

1. Téléchargez Stan Xpress Checkout
2. Dé-zippez and téléversez le dossier wc-stan-checkout dans `/wp-content/plugins/directory`
3. Activez Stan Xpress Checkout depuis la page des plugins

Réglez les configurations de Stan Checkout sur la partie paiements de l'interface WooCommerce

== F.A.Q. ==

= Pourquoi utiliser Stan Xpress Checkout ? =

Stan Xpress Checkout améliore l'expérience de vos utilisateurs à l'achat.

= Comment avoir un identifiant Client et le code secret ? =

Inscrivez vous sur notre site [Stan-app](https://compte.stan-app.fr), les indications vous seront transmises par email.

= Est-ce sécurisé ? =

Stan Xpress Checkout est certifié PCI DSS pour les paiements. Lors de l'achat tous les clients sont authentifiés avec des mots de passe à usage uniques.

= Que se passe t-il si un utilisateur déjà inscrit achète avec Stan Xpress Checkout ? =

Les informations de ce client seront directement remplies lors du parcours d'achat avec Stan Checkout.
